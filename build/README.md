# build

This directory contains the build outputs and runtime assets of the project.

Some files here are generated by the repository's build scripts, while others
are handwritten files directly used by the generated outputs, such as HTML or
CSS assets.

This is notably where the built library bundles, embedded worker/wasm artifacts,
the demo runtime files and the generated documentation site are written.

In most situations, source changes should be performed outside of this
directory, in the corresponding source directories and scripts.
