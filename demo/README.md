# demo

This directory contains the demo application for the Wasp HLS player.

This is not the library exposed by the package itself, but a small React-based
application used to manually test the player and showcase its API.

The code is split into the following directories:

- `src`: source code of the demo application itself.

The demo can be built through the `npm run demo` command and is also rebuilt by
`npm run start` during development.
